# Fresh Cut Barbershop Website

A modern, responsive small-business website built for a local barbershop.
This project was created as a real-world demonstration project for **Swift Designs Studio** to showcase how service businesses can attract customers and accept appointment requests online.

---

## Project Purpose

Many local barbershops rely only on walk-ins, social media, or word of mouth.
The goal of this website is to demonstrate how a professional website can:

* Build trust with new customers
* Showcase services and pricing
* Provide location and contact information
* Allow customers to request appointments online
* Improve online visibility on Google

This project simulates a real client website that can be customized and deployed for actual barbershop businesses.

---

## Features

* Responsive mobile-friendly design
* Service pricing section
* Photo gallery section
* Appointment request form
* Business hours and contact information
* Clean, modern UI for local service businesses

---

## Technologies Used

* HTML5
* CSS3 (Flexbox & Layout)
* Google Fonts
* Basic form handling (front-end)
